# Foldering 최신버전 1.4.3 (260114)

**키보드 중심의 빠른 파일 탐색 및 텍스트 편집 앱**

마우스 없이도 모든 작업을 키보드로 빠르게 수행할 수 있는 파일 관리 및 텍스트 편집 도구입니다.

**Windows & macOS 지원** 🪟🍎

---

## 🚀 빠른 시작

1. 앱 실행 후 `p` 키로 작업 폴더 선택
2. `↑/↓` 키로 파일 탐색
3. `z` 또는 `Enter`로 파일 열기
4. `i` 키로 편집 시작
5. `Ctrl+S`로 저장

---

## ✨ 주요 기능

### 📁 파일 탐색 및 관리
- **키보드 중심**: 모든 작업을 키보드로 수행
- **빠른 검색**: `Ctrl+F` 또는 `/` 키로 파일 검색
- **탭 관리**: 여러 파일을 탭으로 열어 작업 (`Ctrl+←/→`로 탭 이동)
- **파일 작업**: 복사, 이동, 삭제, 이름 변경 (우클릭 메뉴 또는 키보드)
- **되돌리기**: `Ctrl+Z`로 파일 작업 취소

### ✏️ 텍스트 편집
- **즉시 편집**: `i` 키 한 번으로 편집 모드 시작
- **마크다운 지원**: `.md` 파일 자동 렌더링
- **PDF 내보내기**: `Ctrl+P`로 PDF로 저장
- **커스터마이징**: 글씨 크기, 여백 조절 (`Ctrl + +/-`)

### 📝 나만의 메모 (템플릿 기능)
- **템플릿 시스템**: 반복적인 메모를 템플릿으로 관리
- **템플릿 인스턴스**: 템플릿 기반으로 빠르게 메모 작성
- **모드 전환**: `F1` 키로 일반 모드와 나만의 메모 모드 전환
- **상태 저장**: 각 모드의 탭과 경로가 자동으로 저장되어 복원

---

## 🎹 주요 단축키

### 파일 탐색
- `↑/↓`: 파일 목록 이동
- `z` / `Enter`: 파일 선택/열기
- `x` / `Esc`: 뒤로가기
- `←/→`: 이전/다음 파일 보기
- `Ctrl+F` / `/`: 파일 검색

### 파일 편집
- `i`: 편집 모드 시작
- `Ctrl+S`: 저장
- `Esc`: 편집 취소

### 파일 관리
- `n`: 새 파일/폴더 만들기
- `e`: 이름 변경
- `Delete`: 삭제
- `Ctrl+Z`: 되돌리기

### 화면 조작
- `b`: 파일 목록 창 토글
- `p`: 경로 선택
- `o`: 폴더 열기
- `Ctrl+←/→`: 탭 이동

### 기타
- `F1`: 나만의 메모 모드 토글
- `Ctrl+P`: PDF 내보내기
- `Ctrl + +/-`: 글씨 크기 조절

## 📦 설치

### Windows
1. 최신 설치 파일 다운로드
2. 설치 파일 실행
3. 바로 사용 가능

### macOS
1. DMG 파일 다운로드 (Intel Mac: x64, Apple Silicon: arm64)
2. DMG 마운트 후 Applications 폴더로 드래그
3. Applications에서 앱 실행

**⚠️ "damaged and can't be opened" 경고 시:**
터미널에서 실행:
```bash
xattr -d com.apple.quarantine ~/Downloads/Foldering-*.dmg
```
