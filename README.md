# Foldering v1.4.0

**키보드 중심의 빠른 파일 탐색 및 텍스트 편집 앱**

마우스 없이도 모든 작업을 키보드로 빠르게 수행할 수 있는 파일 관리 및 텍스트 편집 도구입니다. 개발자, 작가, 메모 작성자에게 최적화된 경험을 제공합니다.

**이제 macOS를 지원합니다!** 🍎 Intel Mac과 Apple Silicon (M1/M2/M3) 모두 지원합니다.

## 🚀 5분 안에 시작하기

1. 앱을 실행하고 `p` 키를 눌러 작업할 폴더를 선택하세요
2. `↑/↓` 키로 파일 목록을 탐색하세요
3. `z` 또는 `Enter` 키로 파일을 선택하여 내용을 확인하세요
4. `i` 키를 눌러 편집 모드를 시작하세요
5. `Ctrl+S`로 저장하세요
6. `Ctrl+F` 또는 `/` 키로 파일을 검색하세요

## ✨ 무엇을 할 수 있나요?

### 📁 파일 찾기와 관리하기

**키보드로 모든 작업하기**
- 마우스 없이도 모든 작업을 키보드로 수행할 수 있습니다
- 폴더 구조를 직관적으로 탐색할 수 있습니다
- 텍스트 파일을 선택하면 즉시 내용을 확인할 수 있습니다

**원하는 폴더에서 시작하기**
- 시작 경로를 자유롭게 선택할 수 있습니다
- 현재 위치 텍스트를 클릭하면 전체 경로와 폴더명을 전환할 수 있습니다

**빠른 파일 검색**
- `Ctrl+F` 또는 `/` 키로 검색 창을 엽니다
- 현재 폴더만 검색하거나 하위 폴더까지 검색할 수 있습니다
- 입력하는 즉시 검색 결과가 표시됩니다
- 검색어가 파일명에 강조 표시됩니다

### ✏️ 텍스트 편집하기

**즉시 편집 시작**
- `i` 키 한 번으로 편집 모드를 시작합니다
- 마크다운 파일(`.md`)은 자동으로 보기 좋게 표시됩니다

**편안한 읽기와 편집**
- 위/아래 화살표 키로 텍스트를 부드럽게 스크롤할 수 있습니다
- 글씨 크기와 가로 여백을 원하는 대로 조절할 수 있습니다
- `Ctrl + +`로 글씨 크기 증가, `Ctrl + -`로 감소합니다

### 📋 파일 복사, 이동, 삭제하기

**우클릭으로 빠르게 작업하기**
- 파일이나 폴더를 우클릭하면 메뉴가 나타납니다
  - **잘라내기**: 파일이나 폴더를 잘라냅니다
  - **복사**: 파일을 복사합니다 (폴더는 복사할 수 없습니다)
  - **붙여넣기**: 복사하거나 잘라낸 파일을 붙여넣습니다
  - **삭제**: 파일이나 폴더를 삭제합니다

**빈 공간에 붙여넣기**
- 파일 목록의 빈 공간을 우클릭하면 붙여넣기 메뉴가 나타납니다

**실수해도 괜찮아요**
- 파일이나 폴더를 만들거나 삭제하거나 이름을 변경한 후 `Ctrl+Z`로 되돌릴 수 있습니다

### 🎹 자주 쓰는 단축키

**파일 탐색**
- `↑/↓`: 위/아래로 이동
- `z` 또는 `Enter`: 선택/확인
- `x` 또는 `Esc`: 뒤로가기
- `Ctrl+F` 또는 `/`: 파일 검색

**파일 편집**
- `i`: 편집 모드 시작
- `Ctrl+S`: 저장
- `Esc`: 편집 취소

**파일 관리**
- `n`: 새 파일/폴더 만들기
- `e`: 이름 변경
- `Delete`: 삭제
- `Ctrl+Z`: 되돌리기

**화면 이동**
- `←/→`: 이전/다음 파일 보기
- `b`: 파일 목록 창 열기/닫기
- `p`: 경로 선택
- `o`: 폴더 열기

**텍스트 편집기**
- `Ctrl + +`: 글씨 크기 키우기
- `Ctrl + -`: 글씨 크기 줄이기

### ⚙️ 편리한 기능들

**메뉴에서 설정하기**
- 상단 메뉴바의 File 메뉴에서 경로를 선택하거나 폴더를 열 수 있습니다
- Font 메뉴에서 가로 여백과 폰트 크기를 조절할 수 있습니다

**원하는 대로 보기**
- 텍스트 파일만 보이게 필터링할 수 있습니다
- 도움말 패널에서 모든 단축키를 확인할 수 있습니다 (메뉴바 Help > 도움말)
- 라이트 모드와 다크 모드를 전환할 수 있습니다

**안전한 작업**
- 검색 창이나 새 파일 만들기 창이 열려있을 때는 다른 단축키가 작동하지 않아 실수를 방지합니다

## 📦 설치하기

### Windows 사용자

1. 최신 버전의 설치 파일(`Foldering Setup 1.3.1.exe`)을 다운로드하세요
2. 설치 파일을 실행하여 설치를 진행하세요
3. 설치가 완료되면 바로 사용할 수 있습니다

### macOS 사용자 🍎

**Intel Mac (x64) 사용자**:
1. `Foldering-1.3.1-x64.dmg` 파일을 다운로드하세요
2. DMG 파일을 더블 클릭하여 마운트하세요
3. 앱 아이콘을 `Applications` 폴더로 드래그하세요
4. `Applications` 폴더에서 앱을 실행하세요

**Apple Silicon Mac (M1/M2/M3) 사용자**:
1. `Foldering-1.3.1-arm64.dmg` 파일을 다운로드하세요
2. DMG 파일을 더블 클릭하여 마운트하세요
3. 앱 아이콘을 `Applications` 폴더로 드래그하세요
4. `Applications` 폴더에서 앱을 실행하세요

**⚠️ 중요 - "damaged and can't be opened" 경고 해결**:

Chrome 등에서 다운로드한 DMG 파일을 열 때 "damaged and can't be opened" 경고가 나타날 수 있습니다. 이는 macOS Gatekeeper의 보안 기능입니다.

**빠른 해결 방법** (터미널에서 실행):
```bash
# DMG 파일의 quarantine 속성 제거
xattr -d com.apple.quarantine ~/Downloads/Foldering\ App-1.2.0-arm64.dmg
```

**다른 해결 방법**:
- 시스템 설정 → 개인 정보 보호 및 보안 → "확인 없이 열기" 클릭
- 또는 Applications 폴더에서 앱을 우클릭하고 "열기" 선택

## 📖 사용 방법

### 기본 작업 흐름

1. 앱을 실행합니다
2. `p` 키를 눌러 작업할 폴더를 선택합니다
3. `↑/↓` 키로 파일 목록을 탐색합니다
4. `z` 또는 `Enter` 키로 파일을 선택하여 내용을 확인합니다
5. `i` 키를 눌러 편집 모드를 시작합니다
6. 내용을 수정한 후 `Ctrl+S`로 저장합니다

### 파일 검색하기

1. `Ctrl+F` 또는 `/` 키를 눌러 검색 창을 엽니다
2. 찾고 싶은 파일명을 입력합니다 (입력하는 즉시 검색됩니다)
3. "하위 폴더까지 검색" 옵션을 체크하면 하위 폴더까지 검색합니다
4. `↑/↓` 키로 검색 결과를 이동하고 `Enter` 키로 선택합니다
5. `Esc` 키로 검색 창을 닫습니다

### 파일 복사하고 이동하기

1. 파일이나 폴더를 우클릭합니다
2. **잘라내기** 또는 **복사**를 선택합니다 (파일만 복사 가능)
3. 이동하고 싶은 폴더로 이동합니다
4. 빈 공간을 우클릭하고 **붙여넣기**를 선택합니다

### 컨텍스트 메뉴 사용하기

**파일이나 폴더를 우클릭하면:**
- 잘라내기: 파일이나 폴더를 잘라냅니다
- 복사: 파일을 복사합니다 (폴더는 복사할 수 없습니다)
- 붙여넣기: 복사하거나 잘라낸 파일을 붙여넣습니다
- 삭제: 파일이나 폴더를 삭제합니다

**빈 공간을 우클릭하면:**
- 붙여넣기: 복사하거나 잘라낸 파일이 있을 때만 표시됩니다

## 🎯 이번 버전에서 새로워진 것 (v1.4.0)

### ✨ 새로운 기능

#### 템플릿 인스턴스 뷰어 UI 개선
- **템플릿 이름 표시**: 파일 이름 아래에 사용된 템플릿 이름이 표시됩니다
- **확장자 숨김**: 파일 이름에서 `.json` 확장자가 자동으로 숨겨져 더 깔끔하게 표시됩니다
- **파일 이름 강조**: 파일 이름이 더 크고 굵게 표시되어 가독성이 향상되었습니다
- **레이아웃 개선**: 파트 제목과 내용 사이의 간격이 늘어나고, 파트 제목에 구분선이 추가되어 구조가 더 명확해졌습니다

#### 파일 이름 변경 기능 강화
- **F2 키 지원**: 파일을 선택한 상태에서 `F2` 키를 누르면 바로 이름 변경 모드로 진입합니다
- **우클릭 메뉴 추가**: 파일을 우클릭하면 "이름 바꾸기" 옵션이 표시됩니다
- **포커스 자동 복원**: 이름 변경 후 자동으로 파일 탐색기에 포커스가 복원되어 키보드로 계속 작업할 수 있습니다

### 🔧 개선 사항

- **포커스 관리 개선**: 파일 저장, 이름 변경, 파일 선택 해제 후 자동으로 파일 탐색기에 포커스가 복원됩니다
- **탭 키 동작 개선**: 텍스트 편집 모드에서 `Tab` 키를 누르면 탭 문자(`\t`)가 삽입되고, 포커스가 이동하지 않습니다
- **중복 탭 방지**: 같은 파일을 여러 번 열려고 하면 새 탭이 생성되지 않고 기존 탭으로 전환됩니다

### 아키텍처

#### 보안 구조
- **Renderer 프로세스**: React 앱만 실행, Node.js API 직접 접근 불가
- **Main 프로세스**: 파일 시스템 작업 및 메뉴바 관리 수행
- **Preload 스크립트**: `window.api` 객체를 통해 안전한 IPC 통신 인터페이스 제공

#### IPC 통신
- Renderer는 `window.api` 객체를 통해 Main 프로세스와 통신
- 모든 파일 시스템 작업은 Main 프로세스의 서비스 레이어를 통해 수행
- 메뉴바 이벤트는 CustomEvent를 통해 전달

### API 사용법

#### Renderer 프로세스에서 사용

```typescript
// 파일 시스템 작업
const files = await window.api.filesystem.listDirectory(path);
const content = await window.api.filesystem.readFile(filePath);
await window.api.filesystem.writeFile(filePath, content);
await window.api.filesystem.deleteFile(filePath);
await window.api.filesystem.copyFile(sourcePath, destPath);
await window.api.filesystem.moveFile(sourcePath, destPath);

// 파일 검색
const results = await window.api.filesystem.searchFiles(dirPath, query, recursive);

// 경로 관리
const currentPath = await window.api.filesystem.getCurrentDirectory();
await window.api.filesystem.selectStartPath();

// 설정 관리
await window.api.menu.updateCheckbox('hideNonTextFiles', true);
await window.api.menu.updateFontMenu();
```
