# Foldering App v1.1.0 릴리즈 노트

**릴리즈 날짜**: 2025-12-08

## 🎯 주요 변경사항

이번 릴리즈는 성능 최적화와 UI/UX 개선에 중점을 둔 업데이트입니다. 메뉴바 통합, 레이아웃 개선, 그리고 핵심 성능 최적화가 포함되어 있습니다.

## ✨ 새로 추가된 기능

### 메뉴바 통합
- **File 메뉴**: Select Path (`p`), Open Folder (`o`) 옵션 추가
- **Font 메뉴**: 가로 여백 및 폰트 크기 설정을 메뉴바에서 직접 조절 가능
  - 메뉴 항목이 현재 설정값을 자동으로 반영
  - 설정 변경 시 메뉴 체크 상태 즉시 업데이트

### 핫키 추가
- **글씨 크기 조절**: `Ctrl + +` (증가), `Ctrl + -` (감소)로 빠른 조절
  - 도움말 패널에 추가
  - Font 메뉴에 핫키 정보 표시

### UI 개선
- **현재 폴더 이름 표시**: 디렉토리 레이아웃 상단에 현재 폴더 이름만 표시
  - 클릭 시 전체 경로/폴더명 토글 기능
  - 긴 경로에서도 파일 생성 버튼이 항상 보이도록 레이아웃 최적화
- **파일 이름 표시**: 헤더에 선택된 파일 이름만 크게 표시 (`text-lg`)
- **경로 토글 기능**: 현재 위치 텍스트 클릭 시 전체 경로/폴더명 토글

## 🔧 개선사항

### UI 레이아웃 개선
- 파일 편집/삭제/저장 버튼을 헤더로 이동
- 폴더 생성 버튼을 디렉토리 레이아웃 상단으로 이동
- 모든 버튼 스타일 통일 (테마에 맞는 회색 톤)

### 기능 개선
- **디렉토리 변경 시 상태 관리**: 다른 디렉토리로 이동 시 선택된 파일 자동 해제
- **메뉴바 항목 동기화**: Font 메뉴의 체크 상태가 설정 변경 시 즉시 업데이트
- **경로 표시 개선**: 긴 경로에서도 파일 생성 버튼이 항상 보이도록 레이아웃 최적화

### 안정성 개선
- 프로덕션 빌드에서 메뉴바 표시 문제 해결
- 설정 파일 경로 처리 개선 (개발/프로덕션 분리)

## ⚡ 성능 최적화

### 상태 추적 최적화
- Polling 방식에서 콜백 방식으로 변경하여 CPU 사용량 감소
- 100ms interval 제거로 불필요한 리렌더링 방지

### 메모이제이션 적용
- `handleConfigChange` 함수 메모이제이션
- `FileExplorer.loadDirectory` 함수 메모이제이션
- 함수 재생성 방지로 불필요한 리렌더링 감소

### useEffect 최적화
- dependency 배열 최적화
- 불필요한 이벤트 리스너 재등록 방지
- 외부 함수 dependency 제거 (`isHotkey`)

### 전체적인 성능 향상
- 불필요한 리렌더링 감소
- 메모리 사용 최적화
- 반응성 개선

## 🐛 버그 수정

- 프로덕션 빌드에서 메뉴바가 표시되지 않던 문제 해결
- 디렉토리 변경 시 선택된 파일 상태 관리 개선
- 긴 경로에서 파일 생성 버튼이 숨겨지던 문제 해결

## 📦 설치 및 업데이트

### 새로 설치하는 경우
1. `Foldering App Setup 1.1.0.exe` 파일을 다운로드
2. 설치 파일을 실행하여 설치 진행
3. 설치 완료 후 바로 사용 가능

### 기존 버전에서 업데이트하는 경우
1. `Foldering App Setup 1.1.0.exe` 파일을 다운로드
2. 설치 파일을 실행하여 기존 버전 위에 설치
3. 기존 설정 및 데이터는 자동으로 유지됩니다

## 📖 사용 방법

### 메뉴바 사용
- **File > Select Path**: 작업할 폴더 선택 (`p` 키)
- **File > Open Folder**: 새 폴더 열기 (`o` 키)
- **Font > 가로 여백**: 텍스트 편집기의 가로 여백 조절
- **Font > 폰트 크기**: 텍스트 편집기의 폰트 크기 조절 (`Ctrl + +` / `Ctrl + -`)

### 새로운 핫키
- `Ctrl + +`: 글씨 크기 증가
- `Ctrl + -`: 글씨 크기 감소

### UI 변경사항
- 헤더에 선택된 파일 이름이 크게 표시됩니다
- 디렉토리 레이아웃 상단에 현재 폴더 이름이 표시됩니다
- 현재 위치 텍스트를 클릭하면 전체 경로/폴더명이 토글됩니다
- 파일 편집/삭제/저장 버튼이 헤더 우측에 위치합니다
