# Build Assets

이 폴더에는 빌드에 필요한 리소스 파일들이 포함됩니다.

## 아이콘 파일

- `icon.ico`: Windows용 애플리케이션 아이콘 (256x256 권장)

아이콘 파일이 없어도 빌드는 가능하지만, 기본 Electron 아이콘이 사용됩니다.

## 아이콘 생성 방법

1. 온라인 도구 사용:
   - https://convertio.co/kr/png-ico/
   - https://www.icoconverter.com/
   - PNG 이미지를 ICO 형식으로 변환

2. 권장 크기:
   - 256x256 픽셀
   - 투명 배경 지원

3. 파일 위치:
   - `build/icon.ico`에 저장
